# Bookshop test task

## Clone project
```
git clone https://github.com/Roxana1989/bookshop.git
```
## Install gems
```
bundle install
```
## Create database

add to file database.yml

```
username: "Your username"
password: "some password"
```

```
rake db:create
```
## Run migrations

```
rake db:migrate
```
